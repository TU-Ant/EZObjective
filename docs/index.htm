EZ Objectives - Simple Objective Tracking Nodes Are Here!

EZ Objectives (EZO) is a simple, 2 file plugin for Unreal Engine 5. It features designer friendly macros that trivialise the setup of - and access to - a player's progress against a set of objectives.

Simple, easy to read nodes make it a breeze to add new objectives. Events are available to more advanced users.

Ticking off the player's success is also as easy as adding a single, pre-made macro! The library is ready to go for Actor and Level Blueprints.

And with a helpful sample map, you can dive straight into building epic missions. A great resource for prototypes, game jams and smaller projects!
